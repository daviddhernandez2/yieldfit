# Yield Fit

Aplicación web de seguimiento de entrenamientos de fuerza con análisis del progreso al estilo de los rendimientos de activos financieros.

Proyecto final del curso Full Stack Web Development (CEI).

## Stack

- **Backend:** Node.js + Express + MongoDB (Mongoose)
- **Frontend:** React + Vite
- **Autenticación:** JWT + bcrypt
- **Despliegue:** por definir

## Estructura

- `server/` — API REST con Express
- `client/` — Frontend React
